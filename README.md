# Учебная практика: WPF-приложение для управления файлами пользователей (Облачное хранилище)

## 1. Описание проекта
Данный проект представляет собой разработку десктопного приложения на базе технологии WPF (Windows Presentation Foundation) для хранения, загрузки и управления файлами пользователей.

Система имитирует функциональность облачного хранилища, обеспечивая структурированное хранение данных в папках, разграничение прав доступа и ведение журнала действий.

Тип работы: Выпускная квалификационная работа (дипломный проект).

---

## 2. Цель и задачи проекта
**Цель:** разработка клиентского приложения для управления файловой системой пользователей с интеграцией базы данных.

**Задачи:**
- проектирование структуры базы данных
- (таблицы: Пользователи, Файлы, Папки, Права доступа, История действий, Общие ссылки)
- реализация бизнес-логики на C# (классы CloudFile, Folder), разработка интуитивно понятного WPF-интерфейса
- реализация механизмов загрузки/удаления файлов
- расчет занятого пространства
- управление правами доступа и генерация общих ссылок
- тестирование ключевых модулей системы

**Область применения:** 
- локальные корпоративные сети 
- персональное управление данными 
- учебные проекты по архитектуре ПО

---

## 3. Функциональность
- загрузка файлов
- удаление файлов
- создание и навигация по дереву папок
- просмотр списка файлов
- расчет общего занятого дискового пространства

**Ограничения:**
- приложение работает в рамках локальной сети или?? (standalone)
- режима с подключением к СУБД
- тестовый режим эксплуатации


## 4. Архитектура проекта
Тип архитектуры: клиент-сервер

**Технологии:**
 -  C#
 -  WPF (.NET Core/.NET 6+)
 -  PostgreSQL (СУБД)
 -  pgAdmin 4 (для администрирования БД)
 -  Entity Framework Core (ORM)
 -  XAML (разметка интерфейса)

**Структура проекта:**
- src/ — исходный код
- Core/ — бизнес-логика и модели (CloudFile, Folder)
- Data/ — слой доступа к данным и контекст БД, UI/ — WPF-представления и ViewModel
- Tests/ — модульные тесты, Docs/ — документация
